## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.tsx`. The page auto-updates as you edit the file.

Для работы с тестами:
1. Подготовка файловой структуры
1.1. Поместите Question.xlsx в public/
/public
  └── Question.xlsx
1.2. Подготовьте изображения и поместите в формате
/public/images/
  ├── taskImg1.png
  ├── taskImg2.jpg
  ├── taskImg3.jpg
  └── ...
  └── taskImg100.png
